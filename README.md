
# DOOR STATUS
## Hexadecimal code to Data

The program converts the given hex code to informative data of:
- Temperature
- Humidity
- Magnetic Status




## What is used

- Golang

Libraries being used:
- "encoding/hex"
-	"fmt"


## Installation

Run the program using 
```bash
 go run .
```
